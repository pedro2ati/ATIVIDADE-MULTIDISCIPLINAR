# Página de Cadastro Simples

Arquivos incluídos:

- `index.html` — formulário com Nome, Email e Senha
- `styles.css` — estilos básicos
- `script.js` — validação do lado do cliente e mensagem de sucesso

Como abrir:

1. Abra o arquivo `index.html` no navegador (duplo clique) ou
2. Sirva localmente com um servidor HTTP (por exemplo, usando Python):

```powershell
python -m http.server 8000
# então abra http://localhost:8000
```
